# Gmail 每日摘要 Skill

本 Skill 用于自动化处理 Gmail 收件箱中的未读邮件，并将其整理为结构化的中文摘要。

## 核心功能
1. **智能搜索**：严格搜索当前运行时间回溯 24 小时内的未读邮件，确保每日定时任务运行时，覆盖区间与前一日无缝衔接且不重复。
2. **分类整理**：将邮件分为“重要事项”（账单、安全、职位）、“行业资讯”和“营销推广”三类。
3. **中文摘要**：提取发件人、主题、核心内容，并自动将德文、英文等外语邮件翻译为中文。
4. **行动建议**：识别邮件中是否需要用户采取行动（如付款、回复邀请）。

## 使用流程
1. **获取数据**：调用 `gmail_search_messages` 工具获取未读邮件列表。
2. **处理摘要**：使用 `scripts/generate_summary.py` 处理搜索结果。
3. **交付报告**：将生成的 Markdown 报告发送给用户。

## 提示词建议
当用户要求“总结我的邮件”或执行“每日摘要定时任务”时，应遵循以下结构：
- **重要事项优先**：将账单和安全提醒置于最顶端。
- **简明扼要**：每个条目不超过 3 行。
- **多语言处理**：遇到外语邮件时，必须在摘要中包含中文翻译。
